# 요구사항 수정 이력 (CHANGELOG)

본 파일은 [game-requirements.md](./game-requirements.md)의 **모든 변경**을 기록한다.

## 기록 규칙

| 항목            | 규칙                                                                  |
| --------------- | --------------------------------------------------------------------- |
| **언제**        | 변경 반영일 (YYYY-MM-DD)                                              |
| **버전**        | [Semantic Versioning](https://semver.org/) — `MAJOR.MINOR.PATCH`      |
| **유형**        | `Added` · `Changed` · `Removed` · `Deprecated` · `Fixed` · `Proposed` |
| **상세**        | 어떤 REQ/TBD ID가 어떻게 바뀌었는지, 변경 이유                        |
| **문서 동기화** | 본문 수정과 CHANGELOG 작성은 **항상 한 세트**                         |

### 버전 가이드

- **MAJOR**: 범위·코어 루프·시스템 구조 변경 (예: 턴제 → 실시간)
- **MINOR**: 기능·요구사항 추가·확장
- **PATCH**: 문구 정리, 오타, 비모호화 (동작 의미 불변)

### Proposed vs Added

- **Proposed**: 검토 중인 제안. `game-requirements.md` §10에만 있거나 CHANGELOG에만 기록
- **Added/Changed**: 기획 확정 후 본문 반영 시 기록
